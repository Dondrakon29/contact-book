# Contact Book

A simple Python console project for managing contacts.

## Features

- Add new contacts
- Show all contacts
- Search contacts by city
- Delete contacts by number
- Save contacts to JSON file
- Load contacts from JSON file
- Validate empty input fields
- Handle missing JSON file
- Handle broken or empty JSON file
- Basic code refactoring with helper functions

## Technologies

- Python
- JSON
- Git
- GitHub

## How to run

1. Open the project folder.

2. Run the file:

```bash
python Contact_book.py
```

3. Choose an option from the menu:

```text
1 - Add contact
2 - Show contacts
3 - Search by city
4 - Delete contact
0 - Exit
```

## Contact example

Each contact is stored as a dictionary:

```json
{
    "name": "Alex",
    "phone": "+79990000000",
    "city": "Moscow"
}
```
